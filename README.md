# Math RPG - Phase 1a

Phase 1a: Pure frontend, offline, single HTML playable Math RPG game.

## Build Complete
- Single index.html with Phaser.js (CDN), KaTeX (CDN)
- CSS/Canvas drawn school map
- 3 classrooms: 代數房, 幾何房, 統計房
- Quiz system with real P1/P2 questions
- Level + XP system (100*level XP per level)
- HP system (wrong answer = -20 HP)
- localStorage progress saving
- Mobile-friendly virtual joystick
- Web Audio API sound effects
- Dark theme (#1A1A2E)
